# Ideation Agent: Competitor Analyst

Competitive Intelligence Scout & Strategy Analyst for the Ideation Pipeline.

## Installation

```bash
pip install -e .
```

## Usage

```bash
ideation-agent-competitor-analyst run --session-id <session-id>
```
